Simulação de Validação de IP – C#

 Descrição
Programa em C# que simula a validação de acesso por IP, conforme requisito da Entrega 1 do Projeto Interdisciplinar.

 Funcionalidades
- Solicita um IP de origem ao usuário
- Verifica se o IP está em uma lista pré-definida de IPs autorizados
- Exibe mensagem de ACESSO PERMITIDO ou BLOQUEADO
- Registra todas as tentativas em arquivo de log (logs_acesso.txt)


 IPs autorizados 
- 192.168.1.100
- 192.168.1.101
- 10.0.0.50
